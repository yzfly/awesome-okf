# Metadata as Code(kcmd / mdcode)

官方 `agents/mdcode` 工具,把 Dataplex 目录条目(表、术语表、Entry Group)当作本地 YAML+Markdown 管理,做 `pull`/`push` 同步,内置 MCP server。

名字像"代码"但讲的是**元数据即代码**,不是把源代码本身表示成知识——这点的辨析见 [代码支持调研](/docs/code-support-research-zh.md)。
