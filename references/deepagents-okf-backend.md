# deepagents-okf-backend(LangChain Deep Agents 的 OKF 后端)

把 OKF 接进主流 agent 框架的代表项目:为 **LangChain Deep Agents** 提供 OKF 感知的虚拟文件系统后端(已上 PyPI)。把一个 OKF bundle 挂成 agent 的文件系统,按 `type` / `tags` / `title` 做语义查询,且每次写入都校验为合法 OKF。

属于"OKF 作为 agent 框架的知识 / 记忆后端"方向,与做持久记忆的 [hermes-okf](/references/hermes-okf.md) 思路相近,落点在 LangChain 生态。
