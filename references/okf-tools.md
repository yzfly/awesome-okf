# okf-tools(社区 CLI / 库)

OKF(2026-06-12 发布)生态里的社区工具:一套查询、导航、创作 OKF bundle 的 **CLI 与 Python 库**。把 bundle 当成可程序化访问的知识目录来读写。

属早期项目,接口与覆盖度仍在快速变动;更多社区工具(okf-convert、OKFy、okf-skills、okf-builder 等)见 [resources-zh 的「生态工具与转换器」](/docs/resources-zh.md)。
