# W4G1/okf(纯 Rust 零依赖实现)

OKF 生态从 Python 扩展到系统级语言的标志性项目:**纯 Rust、零依赖**的 OKF 实现,生态里第一个用系统级语言做的实现。

它和 [superops-team/okf](/references/superops-okf.md)(Go)、[okftool](/references/okftool.md)(Rust 校验器)一起,标记了 OKF 工具链从脚本语言走向 Go/Rust 多语言落地的信号。

源码:<https://github.com/W4G1/okf>
