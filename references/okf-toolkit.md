# okf-toolkit(Python 全功能 CLI,PyPI)

功能覆盖最全的 **Python** OKF CLI 之一,已发布到 PyPI:`init / new / validate / list / show / index / search / graph / stats`,一套命令覆盖创建、校验、检索、Mermaid 链接图与统计。

与本仓库的 [myokf-cli](/plugins/myokf-cli/) 同属"统一 CLI 入口"形态;更轻量、强调结果可复现的还有 `mikevalstar/okq`,Go 实现的见 [okf-cli](/references/okf-cli.md),均见 [resources-zh](/docs/resources-zh.md)。

源码:<https://github.com/akdira/okf-toolkit>
