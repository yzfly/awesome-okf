# superops-team/okf(Go,项目级知识库)

OKF 的 **Go** 实现之一,定位为 AI Agent 的**项目级知识库**——把一个仓库 / 项目的知识收敛成 OKF bundle 供 agent 使用。

与纯 Rust 的 [W4G1/okf](/references/w4g1-okf.md)、Go 单文件二进制 CLI `agentic-wiki/wiki`、`openknowledge-sh/openknowledge` 同属系统级语言实现这一支,见 [resources-zh](/docs/resources-zh.md)。
