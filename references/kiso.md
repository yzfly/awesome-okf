# kiso(OKF 发布引擎)

OKF 的**发布引擎**:把一个 OKF bundle 渲染成静态网站,定位为 bundle 的"发布 / 阅读端"。

和本仓库的 [okf-to-book](/skills/okf-to-book/SKILL.md)(发布为 VitePress 文档站)、[okf-to-web](/skills/okf-to-web/SKILL.md)(打包成单文件网页)是同一类"consumer / 发布"形态;同方向还有发布/阅读端的 `sljm12/llm_wiki_okf_web` 等,见 [resources-zh](/docs/resources-zh.md)。
