# okf-knowledge(/okf skill,社区星标最高)

社区里星标最高的 OKF 项目之一(~19⭐):一个**可移植的 Claude Code skill**,用一条 `/okf` 命令完成 OKF bundle 的创建、读取、维护与可视化。核心主张和本仓库一致——知识就是可 git 版本化的纯 Markdown,不依赖运行时或数据库。

定位与本仓库的 [okf-creator](/skills/okf-creator/SKILL.md) 相近,属于"用 agent skill 把 OKF 用起来"这一类里被采用最多的实现。完整的 skill / 校验器生态对比见 [resources-zh 的「多语言实现与更多工具」](/docs/resources-zh.md)。
