# knowledge-catalog 仓库(官方)

Google Cloud 的官方仓库,OKF 的总入口。里面有:

- `okf/` —— 规范、参考实现、三个示例 bundle(见 [reference-implementations](/references/reference-implementations.md)、[sample-bundles](/references/sample-bundles.md))
- `agents/` —— 富化 agent、[metadata-as-code](/references/metadata-as-code.md)

官方贡献流程见仓库的 `CONTRIBUTING.md`(需先签 Google CLA)。本仓库向上游提案的记录见 [docs/upstream](/docs/upstream/i18n.en.md)。
